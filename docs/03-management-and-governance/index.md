# Azure Management and Governance roadmap

This domain currently represents **30–35%** of the AZ-900 exam.

## Planned lessons

1. Cost factors and the Azure Pricing Calculator
2. Microsoft Cost Management, budgets, and cost analysis
3. Tags and cost organization
4. Microsoft Purview
5. Azure Policy and policy initiatives
6. Resource locks
7. Azure portal, Cloud Shell, Azure CLI, and Azure PowerShell
8. Azure Arc
9. Infrastructure as code, Azure Resource Manager, ARM templates, and Bicep
10. Azure Advisor
11. Azure Service Health
12. Azure Monitor, Log Analytics, alerts, and Application Insights
13. Exam cheat sheet
14. Practice quiz and answer key

## High-value comparisons

| Requirement | Likely answer |
|---|---|
| Who may perform an action | Azure RBAC |
| What configuration is allowed or required | Azure Policy |
| Prevent accidental change or deletion | Resource lock |
| Recommendations for cost, reliability, security, and performance | Azure Advisor |
| Azure platform incidents affecting you | Azure Service Health |
| Metrics, logs, alerts, and app telemetry | Azure Monitor |
| Terminal in a browser | Azure Cloud Shell |
| Repeatable declarative deployment | ARM template or Bicep |

The lesson list follows the [current official AZ-900 skills measured](https://learn.microsoft.com/credentials/certifications/resources/study-guides/az-900).
