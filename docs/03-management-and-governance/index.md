## High-value comparisons

| Requirement | Likely answer |
|---|---|
| Estimate a design before deployment | Azure Pricing Calculator |
| Analyze actual cloud spending | Microsoft Cost Management / Cost Analysis |
| Notify at a financial target | Budget alert |
| Add business context for cost reporting | Tags |
| Discover, classify, and trace data | Microsoft Purview |
| Who may perform an action | Azure RBAC |
| What configuration is allowed or required | Azure Policy |
| Group several policies | Policy initiative |
| Prevent accidental change or deletion | Resource lock |
| Terminal in a browser | Azure Cloud Shell |
| Extend Azure governance outside Azure | Azure Arc |
| Repeatable declarative deployment | ARM template or Bicep |
| Recommendations for cost, reliability, security, and performance | Azure Advisor |
| Azure platform incidents affecting you | Azure Service Health |
| Metrics, logs, alerts, and app telemetry | Azure Monitor |

## Recommended study approach

For every lesson, answer four questions:

1. What problem does the service solve?
2. At what scope does it operate?
3. What limitation or exam trap matters most?
4. Which neighboring service is the closest distractor?
