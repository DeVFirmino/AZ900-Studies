# Contributing

Thanks for helping improve this AZ-900 study guide.

## Good contributions

- Correct an inaccurate or outdated statement.
- Replace a vague definition with a scenario-based explanation.
- Add an exam clue, comparison, or plausible distractor.
- Add a practice question that tests understanding rather than trivia.

## Style

- Use plain English and short sections.
- Introduce an acronym the first time it appears.
- Prefer current Microsoft product names, such as Microsoft Entra ID.
- Explain why an answer fits and why common alternatives do not.
- Avoid exam dumps, copied exam questions, and claims about guaranteed exam content.
- Link to official Microsoft documentation for facts that may change.

## Pull requests

Keep each pull request focused. Describe what changed, why it is useful, and which official source supports any factual correction.
